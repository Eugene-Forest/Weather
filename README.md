# Weather

该app是一个简易的天气查询软件，实现了在中国范围内的地级行政地区的天气查询功能。



## 密匙须知

在 com.example.weather目录下，有文件名为 SunnyWeatherApplication.kt ,该文件为API密匙定义文件。

### 彩云科技天气API密匙
const val Token="xxxxxxxxxx"
### 高德 Web服务 API密匙
const val key="xxxxxxxxxxxxxxx"
